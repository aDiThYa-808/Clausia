# Thanks for your interest in contributing to this project.

This repository is open source under the Apache 2.0 License, which means you are free to use, fork, and modify the code for your own purposes. However, contributions to this repository are maintained in a curated manner.

## What You Can Contribute Without Permission
### You are welcome to:

- Open issues (bug reports, feature ideas, design feedback)
- Suggest architectural improvements
- Report security or privacy concerns
- Improve documentation or README content

No setup or API keys are required for the above.

---

## Code Contributions

This project integrates with third-party APIs and uses secret keys that are not included in the repository for security and cost reasons. Because of this, pull requests that involve application logic or backend behavior cannot be properly tested without coordination.

### As a result:

- Direct code PRs without prior discussion are not accepted
- All code contributions must be discussed and approved first
